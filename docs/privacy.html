Privacy Policy — AI Ad Block (Starter)

Last updated: August 20, 2025

AI Ad Block (Starter) is a browser extension that blocks advertisements and trackers. It is designed to operate entirely on your device and to minimize data access.

Data Collection

The extension does not collect, store, or transmit personally identifiable information (PII), browsing history, page content, or usage analytics to any server.

Permissions & Why We Need Them

  • declarativeNetRequest (Chromium only): Applies static blocking rules to network requests in the browser.
  • scripting: Injects local content scripts to hide/remove ad elements on the page.
  • storage: Saves extension settings and preferences locally on your device.
  • host_permissions: <all_urls>: Allows the extension to function on any website the user visits.
  • Firefox (MV2): uses a background script instead of a service worker. No remote requests are made by default.

AI / Heuristic Processing

If AI/heuristic features are enabled, any analysis of page structure runs locally in the browser. No page content is uploaded to external services.

Third-Party Sharing

No data is sold, shared, or transferred to third parties.

Children’s Privacy

This extension does not target children and does not knowingly collect data from children.

Changes

We may update this policy from time to time. The latest version will always be available on this page.
